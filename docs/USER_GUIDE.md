## 快速开始

### 1. 启动应用

**启动后端服务：**
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

**启动前端服务：**
```bash
cd frontend
npm install
npm run dev
```

### 2. 访问应用

在浏览器中打开 http://localhost:5173

## 选择数据集

在训练页面左侧，你可以看到"选择数据集"面板。

### 分类数据集
适合学习分类问题：

| 数据集 | 描述 | 难度 |
|--------|------|------|
| XOR | 经典非线性问题，需要隐藏层 | 中等 |
| AND/OR | 线性可分，适合入门 | 简单 |
| Iris | 多分类问题，3类4特征 | 中等 |
| Moons/Circles | 非线性边界可视化 | 中等 |

### 回归数据集
适合学习回归问题：

| 数据集 | 描述 | 难度 |
|--------|------|------|
| Linear | 简单线性拟合 | 简单 |
| Sine | 周期性数据拟合 | 中等 |
| Polynomial | 非线性关系拟合 | 中等 |

## 配置神经网络

### 网络层配置

1. **添加层**：点击"添加层"按钮
2. **删除层**：点击层右上角的删除图标（输入/输出层不能删除）
3. **设置神经元数量**：每层1-100个神经元
4. **选择激活函数**：
   - **ReLU**：最常用，适合隐藏层
   - **Sigmoid**：输出层（二分类）
   - **Softmax**：输出层（多分类）
   - **Linear**：回归问题输出层

### 训练参数

- **优化器**：
  - Adam：推荐，自适应学习率
  - SGD：经典，需要仔细调节学习率

- **损失函数**：
  - Binary Crossentropy：二分类
  - Categorical Crossentropy：多分类
  - MSE：回归问题

- **学习率**：
  - 推荐值：0.001 - 0.01
  - 过大可能导致不收敛
  - 过小收敛太慢

## 训练模型

### 训练流程

1. 选择数据集
2. 配置网络结构
3. 点击"应用配置"
4. 点击"开始训练"

### 训练控制

- **暂停**：暂停训练，可继续
- **继续**：恢复暂停的训练
- **停止**：停止训练
- **重置**：清空训练历史

### 解读训练曲线

**正常训练：**
- Loss 逐渐下降
- Accuracy 逐渐上升
- 训练/验证曲线接近

**过拟合：**
- 训练 Loss 持续下降
- 验证 Loss 开始上升

**欠拟合：**
- Loss 下降很慢或停滞
- Accuracy 很低

## 推理测试

### 使用推理页面

1. 先在训练页面训练模型
2. 切换到推理页面
3. 输入测试数据
4. 点击"预测"

### 理解推理可视化

**逐层激活值**：
- 每个方块代表一个神经元的输出
- 颜色深浅表示激活值大小
- 观察数据如何从输入层流向输出层

## 常见问题

### Q: 训练很慢怎么办？
A: 减小批大小(batch size)或减少神经元数量

### Q: Loss 不下降？
A:
- 检查学习率是否过大或过小
- 尝试增加网络层数或神经元
- 检查损失函数选择是否正确

### Q: 准确率很低？
A:
- 增加训练轮数
- 调整网络结构
- 检查数据集是否正确

### Q: 浏览器崩溃？
A:
- 减小数据集大小
- 减少神经元数量
- 使用更简单的网络结构

## 教学建议

### 教学顺序
1. 从 AND/OR 逻辑门开始（线性可分）
2. 尝试 XOR（理解为什么需要隐藏层）
3. 使用 Iris 数据集（多分类）
4. 尝试回归问题

### 实验建议
1. 观察不同学习率对训练的影响
2. 比较不同激活函数的效果
3. 研究网络深度对性能的影响
4. 观察过拟合现象
